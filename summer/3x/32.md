# 32. Гарантии исключений

## Четыре уровня exception safety

| Гарантия | Что означает |
|---|---|
| **no-throw** (`noexcept`) | никогда не бросает |
| **strong** (строгая) | commit or rollback — либо всё, либо состояние как было |
| **basic** (базовая) | объект валиден, инвариант соблюдён, но состояние могло измениться |
| **no** (отсутствующая) | никаких гарантий, объект может быть сломан |

---

## No-throw — `noexcept`

```cpp
#include <iostream>
#include <string>

// Примитивные типы на стеке — noexcept честный, new не вызывается
void swap_int(int& a, int& b) noexcept {
    int tmp = a; a = b; b = tmp;
}

// Строки — своп через перемещение (только меняем указатели, new нет)
void swap_str(std::string& a, std::string& b) noexcept {
    std::string tmp = std::move(a);
    a = std::move(b);
    b = std::move(tmp);
}

int main() {
    int x = 1, y = 2;
    swap_int(x, y);
    std::cout << x << " " << y << "\n";  // 2 1

    std::string s1 = "hello", s2 = "world";
    swap_str(s1, s2);
    std::cout << s1 << " " << s2 << "\n";  // world hello
}
```

### Типичная гарантия для оператора перемещения

Move constructor и move assignment **всегда** должны быть `noexcept` — иначе `std::vector` при реаллокации будет копировать вместо перемещения (проверяет через `std::move_if_noexcept`):

```cpp
struct Foo {
    Foo(Foo&&) noexcept { }             // вектор будет перемещать — быстро
    Foo& operator=(Foo&&) noexcept { }
};

struct Bar {
    Bar(Bar&&) { }                      // вектор будет копировать — медленно!
    Bar& operator=(Bar&&) { }
};
```

Если `noexcept` поставить, но внутри реально бросит — вызовется `std::terminate`.

## Спецификатор `noexcept`

```cpp
void foo() noexcept;           // никогда не бросает
void bar() noexcept(true);     // то же самое
void baz() noexcept(false);    // может бросать (по умолчанию)

// noexcept(expr) — условный noexcept, удобно в шаблонах
template<typename T>
void swap(T& a, T& b) noexcept(noexcept(T(std::move(a)))) {
    T tmp = std::move(a);
    a = std::move(b);
    b = std::move(tmp);
}
```

## Базовая гарантия через RAII

RAII автоматически даёт базовую гарантию — деструктор вызовется всегда:

```cpp
#include <memory>
#include <stdexcept>

struct Resource {
    std::unique_ptr<int[]> data;
    size_t size;

    Resource(size_t n) : data(std::make_unique<int[]>(n)), size(n) {}

    void resize(size_t new_size) {
        // если make_unique бросит — data и size не изменились
        auto new_data = std::make_unique<int[]>(new_size);
        std::copy(data.get(), data.get() + std::min(size, new_size), new_data.get());
        data = std::move(new_data);  // noexcept — не бросит
        size = new_size;
    }
};
```

## Подвох с `operator=` — порядок операций важен

```cpp
#include <cstring>
#include <stdexcept>

struct MyString {
    char* data;
    size_t size;

    MyString(const char* s) : size(strlen(s)), data(new char[size + 1]) {
        memcpy(data, s, size + 1);
    }
    ~MyString() { delete[] data; }

    // Плохо — нет даже базовой гарантии
    MyString& operator_bad(const MyString& other) {
        delete[] data;                    // удалили старое
        data = new char[other.size + 1]; // бросает — data висячий указатель!
        size = other.size;
        memcpy(data, other.data, size + 1);
        return *this;
    }

    // Базовая гарантия — сначала выделить, потом портить поля
    MyString& operator_basic(const MyString& other) {
        char* new_data = new char[other.size + 1];  // бросает — старое нетронуто
        delete[] data;
        data = new_data;
        size = other.size;
        memcpy(data, other.data, size + 1);
        return *this;
    }

    // Строгая гарантия — copy-and-swap idiom
    MyString& operator=(const MyString& other) {
        MyString tmp(other);         // копируем — бросает — *this нетронут
        std::swap(data, tmp.data);   // noexcept
        std::swap(size, tmp.size);   // noexcept
        return *this;
        // tmp уничтожается — забирает старые данные
    }
};
```

Или ещё короче — принимаем по значению (копирование уже в аргументе):

```cpp
MyString& operator=(MyString other) {  // копия уже сделана
    std::swap(data, other.data);
    std::swap(size, other.size);
    return *this;
}
```

## Строгая гарантия для `push_back`

```cpp
#include <memory>
#include <stdexcept>
#include <algorithm>

template<typename T>
struct MyVector {
    std::unique_ptr<T[]> data;
    size_t size = 0;
    size_t capacity = 0;

    void push_back(const T& value) {
        if (size == capacity) {
            size_t new_cap = capacity == 0 ? 1 : capacity * 2;
            auto new_data = std::make_unique<T[]>(new_cap);  // бросает — старое нетронуто

            try {
                std::copy(data.get(), data.get() + size, new_data.get()); // копируем
                new_data[size] = value;                                    // добавляем
            } catch (...) {
                // new_data уничтожится автоматически (unique_ptr)
                throw;  // старый буфер нетронут — строгая гарантия
            }

            data = std::move(new_data);
            capacity = new_cap;
        } else {
            data[size] = value;
        }
        size++;
    }
};
```

Если move constructor `noexcept` — можно перемещать вместо копирования (быстрее):

```cpp
// вместо std::copy используем move_if_noexcept
for (size_t i = 0; i < size; i++)
    new_data[i] = std::move_if_noexcept(data[i]);
```
